This program uses below technologies:

1. Java (JEE, Spring)
2. Javascript (MAngular JS)
3.	Bootstrap
4.	RESTful
5.	MongoDB
6.	Maven
7. Apache Tomcat

The front end was implemented by html and angular JS. And some UI widgets of Bootstrap was also used.
Server side component was developed with Java and Angular support for Spring Restful wss used for the communication between the client and server.
Student data is stored in a MongoDB where the db connection is configured by Spring Data.
The web appliaction was tested in Apache Tomcat server.
The project is configured as  a Maven project.
